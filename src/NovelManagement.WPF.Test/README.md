# 小说管理系统 - 界面功能测试项目

## 📖 项目简介

这是小说管理系统的界面功能测试项目，用于验证WPF界面的基本功能和用户体验。

## 🚀 快速开始

### 运行测试项目

```bash
# 构建项目
dotnet build src/NovelManagement.WPF.Test/NovelManagement.WPF.Test.csproj

# 运行项目
dotnet run --project src/NovelManagement.WPF.Test/NovelManagement.WPF.Test.csproj
```

### 测试界面

启动应用程序后，您可以测试以下界面：

1. **装备体系管理** - 点击"装备体系"按钮
   - 查看装备类型列表
   - 测试搜索和筛选功能
   - 点击装备项查看详情

2. **功法体系管理** - 点击"功法体系"按钮
   - 浏览功法列表
   - 测试类别筛选
   - 查看功法详细信息

3. **商业体系管理** - 点击"商业体系"按钮
   - 查看商业模式列表
   - 测试功能按钮

4. **时间线管理** - 点击"时间线"按钮
   - 查看时间线可视化
   - 测试事件管理功能
